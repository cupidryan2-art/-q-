    模块与生态系统

    Python 真正的超能力不是语言本身——是生态系统。标准库覆盖了几乎所有日常任务，PyPI（Python Package Index）上还有超过 50 万个第三方包。 这一章教你怎么在这片丛林里找到方向。

    import 系统

    Python 把代码组织成模块（module）——单个 .py 文件——和包（package）——包含 __init__.py 文件的目录。用 import 把它们引入你的命名空间。

    Python — import 的几种写法
    # 1. 导入整个模块——通过 模块名.名字 访问
import math
math.sqrt(16)     # 4.0
math.pi           # 3.141592653589793

# 2. 导入特定名字——引入到本地命名空间
from math import sqrt, pi
sqrt(16)          # 4.0（不需要 math. 前缀）
pi                # 3.141592653589793

# 3. 带别名（alias）导入——对长名字必不可少
import numpy as np          # 行业标准别名
import pandas as pd
import matplotlib.pyplot as plt

np.array([1, 2, 3])

# 4. 导入全部——在生产代码里避免这样写
from math import *    # 污染命名空间——难以追踪名字从哪来的

# 5. 条件导入——用于可选依赖
try:
    import ujson as json    # 如果有更快的 JSON 库就用它
except ImportError:
    import json             # 回退到标准库
    C++ 对比

    import math ≈ #include <cmath>，但 import 在运行时发生——第一次导入模块时，该模块的代码会真正执行。之后再导入同一个模块会从 sys.modules 缓存里直接返回，类似 include guard。

    from math import sqrt 大致相当于在包含头文件后写 using std::sqrt;。

    import numpy as np 类似 namespace np = numpy;——你只是为模块对象创建了一个短别名。

    Python 怎么找到模块

    Python — 模块搜索路径
    import sys

# Python 按顺序搜索这些目录：
for path in sys.path:
    print(path)

# 1. 被运行脚本所在的目录（REPL/当前目录用 ''）
# 2. PYTHONPATH 环境变量里的目录
# 3. 标准库目录
# 4. site-packages（pip 安装第三方包的地方）

    标准库精选

    Python 的标准库大到有经验的开发者也要花好几年才能摸清每个角落。下面是你在数据和脚本工作里最常用的模块。

    math — 数学函数

    Python
    import math

math.sqrt(2)         # 1.4142135623730951
math.floor(3.7)      # 3
math.ceil(3.2)       # 4
math.log(math.e)     # 1.0  — 自然对数
math.log10(1000)     # 3.0
math.log2(8)         # 3.0
math.sin(math.pi/2)  # 1.0
math.factorial(10)   # 3628800
math.gcd(48, 18)     # 6
math.isfinite(float('inf'))  # False
math.isnan(float('nan'))     # True

    random — 伪随机数

    Python
    import random

random.random()              # [0.0, 1.0) 之间的 float
random.randint(1, 6)         # [1, 6] 之间的整数（含两端）——掷骰子
random.uniform(0.5, 1.5)     # [0.5, 1.5] 之间的 float

items = ["苹果", "香蕉", "樱桃"]
random.choice(items)         # 随机取一个元素
random.choices(items, k=3)   # 有放回地随机取 3 个
random.sample(items, k=2)    # 无放回地随机取 2 个
random.shuffle(items)        # 就地随机打乱

# 可复现的随机性——数据科学实验必备
random.seed(42)
print(random.random())   # 对 seed=42 永远是同一个值

    datetime — 日期与时间

    Python
    from datetime import datetime, date, timedelta

now = datetime.now()
print(now)                          # 2024-03-15 14:32:07.123456

today = date.today()
print(today)                        # 2024-03-15

# 解析日期字符串
d = datetime.strptime("2024-01-15", "%Y-%m-%d")

# 格式化为字符串
d.strftime("%Y年%m月%d日")          # '2024年01月15日'

# 日期运算
one_week = timedelta(weeks=1)
next_week = today + one_week

# 两个日期之差
delta = datetime(2025, 1, 1) - datetime.now()
print(f"距离 2025 年还有 {delta.days} 天")

    pathlib — 文件系统

    Python
    from pathlib import Path   # 现代面向对象的方式（推荐）

p = Path("data/results.csv")
p.exists()          # True / False
p.name              # 'results.csv'
p.stem              # 'results'
p.suffix            # '.csv'
p.parent            # Path('data')

# 安全地拼接路径（在 Windows 和 Unix 上都能用）
data_dir = Path("data")
output   = data_dir / "output" / "report.csv"   # / 运算符！

# 遍历目录
for f in Path(".").iterdir():
    if f.suffix == ".py":
        print(f)

# 创建目录
Path("logs").mkdir(parents=True, exist_ok=True)

    pip 与虚拟环境

    pip 是 Python 的包安装工具，从 PyPI 下载并安装包。永远在虚拟环境里用它，让不同项目保持隔离。

    终端 — 常用 pip 命令
    # 安装包
pip install numpy

# 安装特定版本
pip install numpy==1.26.4

# 安装多个包
pip install numpy pandas matplotlib

# 从 requirements 文件安装
pip install -r requirements.txt

# 升级包
pip install --upgrade pandas

# 卸载
pip uninstall numpy

# 列出已安装的包
pip list

# 查看某个包的信息
pip show pandas

# 把当前环境冻结到 requirements 文件
pip freeze > requirements.txt
    终端 — 虚拟环境工作流
    # 创建
python3 -m venv .venv

# 激活
source .venv/bin/activate    # macOS / Linux
.venv\Scripts\activate       # Windows

# 提示符会变成：(.venv) $

# 安装包——它们进入 .venv/lib/，不影响系统 Python
pip install pandas matplotlib scikit-learn

# 保存依赖
pip freeze > requirements.txt

# 完成后退出
deactivate

# 在另一台机器上重建环境：
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
    现代替代方案
    uv（来自 Astral）是用 Rust 写的极速现代替代品，取代 pip + venv。uv init、uv add pandas、uv run script.py。如果你在 2024 年后开始新项目，值得了解一下。

    数据科学生态预览

    四个库构成了 Python 数据科学的骨架。你会在第 7–11 章里和它们深度相处。先认识一下它们各自的定位：

    终端 — 安装整套工具
    pip install numpy pandas matplotlib scikit-learn

    NumPy — 数值计算

    NumPy 提供 ndarray——一个固定类型、C 语言底层的多维数组。它是其他一切的基础。关键洞见：NumPy 操作在编译好的 C/Fortran 代码里跑，比 Python 循环快几个数量级。

    Python — NumPy 味道
    import numpy as np

a = np.array([1, 2, 3, 4, 5])
a * 2             # array([2, 4, 6, 8, 10]) — 向量化，不需要循环！
a ** 2            # array([1, 4, 9, 16, 25])
a.mean()          # 3.0
a.std()           # 1.4142...

# 2D 数组——矩阵操作
m = np.array([[1, 2], [3, 4]])
m.T               # 转置
m @ m             # 矩阵乘法（Python 3.5+ 的 @ 运算符）

    pandas — 数据分析

    pandas 提供 DataFrame——一张带标签列和行索引的表格。把它想成你可以编程的电子表格。第 3 章里的字典列表是你的原始数据，pandas DataFrame 是它长大后的样子。

    Python — pandas 味道
    import pandas as pd

df = pd.read_csv("sales.csv")   # 把 CSV 加载成 DataFrame

df.head()                    # 前 5 行
df.info()                    # 列类型和空值计数
df.describe()                # 汇总统计

df["revenue"].mean()         # 列均值
df[df["revenue"] > 1000]     # 过滤行
df.groupby("region").sum()   # SQL 风格的 GROUP BY

    Matplotlib — 数据可视化

    Matplotlib 是 Python 的基础绘图库。用起来可能有点啰嗦，但很精确。更高层的库如 seaborn 和 pandas 自带的 .plot() 都是对它的封装。

    Python — matplotlib 味道
    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

plt.plot(x, y, marker='o')
plt.title("平方数")
plt.xlabel("x")
plt.ylabel("x²")
plt.savefig("squares.png")   # 或者在 Jupyter 里用 plt.show()

    scikit-learn — 机器学习

    scikit-learn 是经典算法的标准 ML 库：回归、分类、聚类、降维、模型评估。它有一套极其一致的 API：每个模型都有 .fit(X, y)、.predict(X) 和 .score(X, y)。

    Python — scikit-learn 味道
    from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

model = LinearRegression()
model.fit(X_train, y_train)          # 训练
predictions = model.predict(X_test)  # 推断
score = model.score(X_test, y_test)  # R² 分数
