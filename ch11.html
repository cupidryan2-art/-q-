    Data Visualization 数据可视化

    A chart communicates in seconds what a table takes minutes to parse. This chapter covers matplotlib's building blocks, then seaborn's statistical shorthand — and closes with a practical guide to choosing the right chart type for your data.

    Terminal — install
    pip install matplotlib seaborn

    Matplotlib Basics

    matplotlib is Python's foundational plotting library. Its pyplot interface mimics MATLAB's state-machine API — you build a chart by calling functions that modify a global "current figure." This is fine for quick plots; for production-quality figures you'll want the object-oriented API (covered next).

    Python — the four core chart types
    import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)

# Line chart (折线图)
plt.plot(x, y)
plt.title("Sine Wave")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.show()   # or plt.savefig("sine.png")

# Scatter plot (散点图)
rng = np.random.default_rng(42)
x2 = rng.normal(0, 1, 200)
y2 = x2 * 0.8 + rng.normal(0, 0.5, 200)
plt.scatter(x2, y2, alpha=0.5, color="steelblue", s=20)
plt.title("Scatter: x vs y")
plt.show()

# Bar chart (柱状图)
depts  = ["Engineering", "Sales", "HR", "Product"]
counts = [42, 28, 15, 23]
plt.bar(depts, counts, color="#CC785C")
plt.title("Headcount by Department")
plt.xticks(rotation=30)
plt.show()

# Histogram (直方图)
data = rng.normal(70, 15, 500)
plt.hist(data, bins=30, edgecolor="white", color="steelblue")
plt.title("Score Distribution")
plt.xlabel("Score")
plt.ylabel("Frequency")
plt.show()

    The Object-Oriented API: Figure and Axes

    Every matplotlib plot consists of a Figure (图形) — the outer container — and one or more Axes (坐标系) objects — the individual plot areas. The OO API gives you explicit control over each element and is essential for subplots and publication-quality output.

    Python — Figure and Axes
    import matplotlib.pyplot as plt
import numpy as np

# Create Figure and Axes explicitly
fig, ax = plt.subplots(figsize=(8, 5))   # figsize in inches

x = np.linspace(0, 10, 100)
ax.plot(x, np.sin(x), label="sin(x)",  color="#CC785C",  linewidth=2)
ax.plot(x, np.cos(x), label="cos(x)",  color="steelblue", linewidth=2, linestyle="--")

# Every label, title, tick is set on the Axes object
ax.set_title("Sine and Cosine",   fontsize=14, fontweight="bold")
ax.set_xlabel("x",               fontsize=11)
ax.set_ylabel("y",               fontsize=11)
ax.legend(fontsize=10)
ax.grid(True, alpha=0.3)
ax.set_xlim(0, 10)
ax.set_ylim(-1.5, 1.5)

fig.tight_layout()   # prevent label clipping
plt.show()

    Multiple subplots (子图)

    Python — subplots grid
    fig, axes = plt.subplots(2, 2, figsize=(10, 8))
# axes is a 2×2 NumPy array of Axes objects

rng = np.random.default_rng(0)
data = rng.normal(50, 15, 300)

axes[0, 0].hist(data, bins=20, color="#CC785C", edgecolor="white")
axes[0, 0].set_title("Histogram")

axes[0, 1].boxplot(data, vert=True)
axes[0, 1].set_title("Boxplot")

axes[1, 0].scatter(range(len(data)), data, alpha=0.4, s=10)
axes[1, 0].set_title("Scatter (index vs value)")

axes[1, 1].plot(sorted(data))
axes[1, 1].set_title("Sorted values")

fig.suptitle("Distribution Explorer", fontsize=15, fontweight="bold")
fig.tight_layout()
plt.savefig("distribution_report.png", dpi=150, bbox_inches="tight")
    Mental Model
    Think of Figure as a canvas (sf::RenderWindow if you know SFML) and each Axes as a viewport into that canvas. You can have multiple viewports with independent coordinate systems. plt.subplots(2, 2) creates a 2×2 grid of viewports for you.

    Styling and Customisation

    Python — colours, markers, annotations
    fig, ax = plt.subplots(figsize=(8, 5))

# Colours: named, hex, RGB tuple
ax.plot([1,2,3], color="tomato")
ax.plot([1,2,3], color="#2E86AB")
ax.plot([1,2,3], color=(0.2, 0.5, 0.8))

# Line styles and markers
ax.plot(x, y, linestyle="-",   marker="o", markersize=4)  # solid + circles
ax.plot(x, y, linestyle="--",  marker="s", markersize=6)  # dashed + squares
ax.plot(x, y, linestyle=":",   marker="^")                # dotted + triangles
ax.plot(x, y, "r--o")    # shorthand: red dashed with circles

# Fill between two lines
ax.fill_between(x, y1, y2, alpha=0.2, color="steelblue")

# Annotations
ax.annotate("Peak", xy=(1.57, 1.0), xytext=(2.5, 0.8),
            arrowprops=dict(arrowstyle="->", color="gray"),
            fontsize=10)

# Global style sheets
plt.style.use("seaborn-v0_8-whitegrid")   # clean academic style
# Other options: "ggplot", "bmh", "dark_background", "seaborn-v0_8-poster"

    Seaborn — Statistical Visualisation (统计可视化)

    seaborn is built on matplotlib and provides higher-level functions for statistical plots. It understands pandas DataFrames natively — you pass column names as strings and it handles the rest. Seaborn also has better default aesthetics.

    Python — seaborn essentials
    import seaborn as sns
import pandas as pd
import matplotlib.pyplot as plt

# seaborn works directly with DataFrames — specify column names as strings
df = pd.read_csv("titanic.csv")   # classic dataset

# Distribution plot
sns.histplot(df, x="age", kde=True, hue="survived")  # kde = density curve overlay

# Box plot (箱线图) — distribution by group
sns.boxplot(df, x="class", y="fare", hue="survived", palette="muted")

# Violin plot — boxplot + distribution shape
sns.violinplot(df, x="class", y="age")

# Count plot — categorical bar chart
sns.countplot(df, x="class", hue="survived")

# Scatter with regression line
sns.regplot(df, x="age", y="fare", scatter_kws={"alpha":0.3})

# Line plot with confidence interval (bootstrapped)
sales = pd.read_csv("monthly_sales.csv")
sns.lineplot(sales, x="month", y="revenue", hue="region")

    Heatmap for correlation matrices

    Python — correlation heatmap
    import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

df = pd.read_csv("employees.csv")

# Compute correlation matrix (only numeric columns)
corr = df.select_dtypes(include="number").corr()

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(
    corr,
    annot=True,        # show values in cells
    fmt=".2f",         # 2 decimal places
    cmap="coolwarm",   # diverging colormap: blue=negative, red=positive
    vmin=-1, vmax=1,   # fix scale to [-1, 1]
    linewidths=0.5,
    ax=ax,
)
ax.set_title("Feature Correlation Matrix")
fig.tight_layout()
plt.savefig("correlation.png", dpi=150, bbox_inches="tight")

    Pair plot — all pairwise relationships at once

    Python
    import seaborn as sns

df = pd.read_csv("iris.csv")   # famous 4-feature flower dataset

# Creates an n×n grid of scatter plots + diagonal histograms
# hue colours each point by class — instantly shows separability
g = sns.pairplot(df, hue="species", diag_kind="kde", plot_kws={"alpha": 0.5})
g.fig.suptitle("Iris: All Pairwise Feature Relationships", y=1.02)
plt.savefig("pairplot.png", dpi=120, bbox_inches="tight")

    Choosing the Right Chart Type (图表类型选择)

    Question you're askingBest chartFunction
    How is this variable distributed?Histogram / KDEsns.histplot, sns.kdeplot
    Distribution across categories?Box / Violin plotsns.boxplot, sns.violinplot
    Relationship between two continuous variables?Scatter plotplt.scatter, sns.scatterplot
    Linear trend between two variables?Regression plotsns.regplot
    How do categories compare in size?Bar chartplt.bar, sns.barplot
    Change over time?Line chartplt.plot, sns.lineplot
    Correlation between many variables?Heatmapsns.heatmap
    All pairwise relationships?Pair plotsns.pairplot
    Part-to-whole composition?Stacked bar / Pieplt.pie (use sparingly)

    Saving Figures (保存图表)

    Python — plt.savefig
    fig, ax = plt.subplots()
ax.plot([1, 2, 3], [4, 5, 6])

# Save before plt.show() — show() clears the figure
fig.savefig("chart.png",                    # PNG (raster)
            dpi=300,                         # 300 dpi = print quality
            bbox_inches="tight",             # don't clip labels
            facecolor="white",               # white background
            transparent=False)

fig.savefig("chart.pdf", bbox_inches="tight")   # PDF (vector — scales losslessly)
fig.savefig("chart.svg", bbox_inches="tight")   # SVG (vector — editable in Illustrator)

# In Jupyter notebooks — just plt.show() or leave the figure as last cell output
# For inline display: %matplotlib inline  (put in first cell)
    Jupyter Tip
    In a Jupyter notebook, add %matplotlib inline in your first cell to display plots inline. For interactive zoom/pan, use %matplotlib widget (requires ipympl). For reports and presentations, save as PDF or SVG — they scale losslessly and look sharp at any size.
