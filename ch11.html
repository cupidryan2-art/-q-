    数据可视化 Data Visualization

    一张图表传递的信息，往往胜过一张密密麻麻的数据表。这一章先讲 matplotlib 的基础骨架，再介绍 seaborn 的统计图表高级接口——最后给你一份实用的"选图指南"，告诉你在什么场景下该选什么类型的图。

    终端 — 安装
    pip install matplotlib seaborn

    matplotlib 基础

    matplotlib 是 Python 可视化的"地基"，历史悠久、功能强大。它的 pyplot 接口模仿了 MATLAB 的风格——调用一系列函数来修改一个全局的"当前图形"状态机。快速出图用这个接口就够了；如果你需要精细控制多子图布局，就要用面向对象的 API（下一节介绍）。

    Python — 四种核心图表类型
    import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)

# 折线图 (Line chart)
plt.plot(x, y)
plt.title("正弦波")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.show()   # 或者 plt.savefig("sine.png")

# 散点图 (Scatter plot)
rng = np.random.default_rng(42)
x2 = rng.normal(0, 1, 200)
y2 = x2 * 0.8 + rng.normal(0, 0.5, 200)
plt.scatter(x2, y2, alpha=0.5, color="steelblue", s=20)
plt.title("散点图：x vs y")
plt.show()

# 柱状图 (Bar chart)
depts  = ["工程", "销售", "HR", "产品"]
counts = [42, 28, 15, 23]
plt.bar(depts, counts, color="#CC785C")
plt.title("各部门人数")
plt.xticks(rotation=30)
plt.show()

# 直方图 (Histogram)
data = rng.normal(70, 15, 500)
plt.hist(data, bins=30, edgecolor="white", color="steelblue")
plt.title("分数分布")
plt.xlabel("分数")
plt.ylabel("频次")
plt.show()

    面向对象 API：Figure 与 Axes

    每张 matplotlib 图都由两个层级组成：Figure（整张画布）和 Axes（画布上的一个绘图区域）。面向对象 API 让你可以精确控制每一个元素，对多子图和需要发表级别输出的场景来说不可或缺。

    Python — Figure 与 Axes
    import matplotlib.pyplot as plt
import numpy as np

# 显式创建 Figure 和 Axes
fig, ax = plt.subplots(figsize=(8, 5))   # figsize 单位是英寸

x = np.linspace(0, 10, 100)
ax.plot(x, np.sin(x), label="sin(x)",  color="#CC785C",  linewidth=2)
ax.plot(x, np.cos(x), label="cos(x)",  color="steelblue", linewidth=2, linestyle="--")

# 所有标签、标题、刻度都在 Axes 对象上设置
ax.set_title("正弦与余弦",   fontsize=14, fontweight="bold")
ax.set_xlabel("x",           fontsize=11)
ax.set_ylabel("y",           fontsize=11)
ax.legend(fontsize=10)
ax.grid(True, alpha=0.3)
ax.set_xlim(0, 10)
ax.set_ylim(-1.5, 1.5)

fig.tight_layout()   # 防止标签被截断
plt.show()

    多子图 (subplots)

    Python — 2×2 子图网格
    fig, axes = plt.subplots(2, 2, figsize=(10, 8))
# axes 是一个 2×2 的 NumPy 数组，每个元素都是一个 Axes 对象

rng = np.random.default_rng(0)
data = rng.normal(50, 15, 300)

axes[0, 0].hist(data, bins=20, color="#CC785C", edgecolor="white")
axes[0, 0].set_title("直方图")

axes[0, 1].boxplot(data, vert=True)
axes[0, 1].set_title("箱线图")

axes[1, 0].scatter(range(len(data)), data, alpha=0.4, s=10)
axes[1, 0].set_title("散点图（索引 vs 值）")

axes[1, 1].plot(sorted(data))
axes[1, 1].set_title("排序后的值")

fig.suptitle("分布探索", fontsize=15, fontweight="bold")
fig.tight_layout()
plt.savefig("distribution_report.png", dpi=150, bbox_inches="tight")
    C++ 类比
    把 Figure 想成一张画布（类似 SFML 里的 sf::RenderWindow），每个 Axes 是这张画布上的一个视口，有自己独立的坐标系。plt.subplots(2, 2) 就是帮你在画布上划好 2×2 四个视口。

    样式与自定义

    Python — 颜色、标记与注释
    fig, ax = plt.subplots(figsize=(8, 5))

# 颜色：名称、十六进制、RGB 元组都支持
ax.plot([1,2,3], color="tomato")
ax.plot([1,2,3], color="#2E86AB")
ax.plot([1,2,3], color=(0.2, 0.5, 0.8))

# 线型与标记点
ax.plot(x, y, linestyle="-",   marker="o", markersize=4)  # 实线 + 圆点
ax.plot(x, y, linestyle="--",  marker="s", markersize=6)  # 虚线 + 方块
ax.plot(x, y, linestyle=":",   marker="^")                # 点线 + 三角
ax.plot(x, y, "r--o")    # 简写：红色虚线 + 圆点

# 在两条线之间填色
ax.fill_between(x, y1, y2, alpha=0.2, color="steelblue")

# 添加注释（带箭头）
ax.annotate("峰值", xy=(1.57, 1.0), xytext=(2.5, 0.8),
            arrowprops=dict(arrowstyle="->", color="gray"),
            fontsize=10)

# 全局样式表
plt.style.use("seaborn-v0_8-whitegrid")   # 干净的学术风格
# 其他可选："ggplot"、"bmh"、"dark_background"、"seaborn-v0_8-poster"

    seaborn — 统计可视化神器

    seaborn 构建在 matplotlib 之上，专门为统计图表设计了更高级的接口。它原生理解 pandas DataFrame——你只需要传列名字符串，它自己处理剩下的事情。而且默认配色和布局比 matplotlib 好看多了。

    Python — seaborn 核心用法
    import seaborn as sns
import pandas as pd
import matplotlib.pyplot as plt

# seaborn 直接吃 DataFrame，列名传字符串即可
df = pd.read_csv("titanic.csv")   # 经典数据集

# 分布图（带核密度估计曲线）
sns.histplot(df, x="age", kde=True, hue="survived")

# 箱线图 — 按分组展示分布
sns.boxplot(df, x="class", y="fare", hue="survived", palette="muted")

# 小提琴图 — 箱线图 + 分布形状
sns.violinplot(df, x="class", y="age")

# 计数图 — 分类变量的柱状图
sns.countplot(df, x="class", hue="survived")

# 散点图 + 回归线
sns.regplot(df, x="age", y="fare", scatter_kws={"alpha":0.3})

# 折线图（自动计算置信区间）
sales = pd.read_csv("monthly_sales.csv")
sns.lineplot(sales, x="month", y="revenue", hue="region")

    用热力图展示相关矩阵

    Python — 相关性热力图
    import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

df = pd.read_csv("employees.csv")

# 计算相关矩阵（只取数值列）
corr = df.select_dtypes(include="number").corr()

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(
    corr,
    annot=True,        # 在格子里显示数值
    fmt=".2f",         # 保留两位小数
    cmap="coolwarm",   # 发散色图：蓝=负相关，红=正相关
    vmin=-1, vmax=1,   # 固定色标范围
    linewidths=0.5,
    ax=ax,
)
ax.set_title("特征相关性矩阵")
fig.tight_layout()
plt.savefig("correlation.png", dpi=150, bbox_inches="tight")

    Pair plot — 一次看完所有两两关系

    Python
    import seaborn as sns

df = pd.read_csv("iris.csv")   # 经典鸢尾花数据集，4 个特征

# 生成 n×n 散点图网格，对角线是各特征的分布图
# hue 按类别着色，一眼就能看出特征的可分性
g = sns.pairplot(df, hue="species", diag_kind="kde", plot_kws={"alpha": 0.5})
g.fig.suptitle("Iris：所有特征的两两关系", y=1.02)
plt.savefig("pairplot.png", dpi=120, bbox_inches="tight")

    怎么选图表类型？

    你想回答的问题最佳图表函数
    某个变量是怎么分布的？直方图 / KDEsns.histplot、sns.kdeplot
    各分组的分布有何不同？箱线图 / 小提琴图sns.boxplot、sns.violinplot
    两个连续变量之间有关系吗？散点图plt.scatter、sns.scatterplot
    两个变量之间有线性趋势吗？回归图sns.regplot
    各类别的数量/大小对比？柱状图plt.bar、sns.barplot
    随时间的变化趋势？折线图plt.plot、sns.lineplot
    很多变量之间的相关性？热力图sns.heatmap
    所有特征的两两关系？Pair plotsns.pairplot
    部分与整体的占比？堆叠柱状图 / 饼图plt.pie（谨慎使用）

    保存图表

    Python — plt.savefig
    fig, ax = plt.subplots()
ax.plot([1, 2, 3], [4, 5, 6])

# 必须在 plt.show() 之前保存——show() 会清空图形
fig.savefig("chart.png",                    # PNG（位图）
            dpi=300,                         # 300 dpi = 印刷级分辨率
            bbox_inches="tight",             # 自动裁剪，不截断标签
            facecolor="white",               # 白色背景
            transparent=False)

fig.savefig("chart.pdf", bbox_inches="tight")   # PDF（矢量，无限缩放不模糊）
fig.savefig("chart.svg", bbox_inches="tight")   # SVG（矢量，可用 Illustrator 编辑）

# Jupyter Notebook 里：直接 plt.show() 或把图形作为最后一行输出
# 开启内联显示：%matplotlib inline（放在第一个 cell）
    Jupyter 小贴士
    在 Jupyter Notebook 里，在第一个 cell 写 %matplotlib inline 就能让图表直接嵌入输出。想要交互式缩放/平移，用 %matplotlib widget（需要安装 ipympl）。写报告或做演讲，存成 PDF 或 SVG——矢量格式任意缩放都清晰。
