    第 2 章

    控制流

    Python 的控制流看起来很熟悉——if、for、while 都有。区别在于什么东西替代了花括号：缩进。然后在最后还藏着一个 Python 真正的惊喜：列表推导式。

    缩进即结构

    C++ 里，花括号 {} 定义代码块。缩进只是可选的——是靠 linter 强制执行的约定。 Python 里，缩进就是语法本身。没有花括号。解释器用空白符来判断代码块的边界。

    C++ — 花括号定义代码块
    if (x > 0) {
    std::cout << "正数" << std::endl;
    std::cout << "还在代码块里" << std::endl;
}
std::cout << "代码块外面" << std::endl;
    Python — 缩进定义代码块
    if x > 0:
    print("正数")
    print("还在代码块里")   # 同样的缩进 = 同一个代码块
print("代码块外面")         # 回到第 0 列 = 退出代码块
    规则
    每级缩进用 4 个空格。永远不要混用 tab 和空格——Python 3 会抛 TabError。配置你的编辑器在按 Tab 时插入空格，现代 Python 编辑器默认都这样。

    if / elif / else

    注意每个条件后面的冒号 :——这不是可选的，它是语法的一部分，标志着缩进代码块的开始。

    Python
    score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"等级：{grade}")   # 等级：B

    真值（truthiness）

    Python 在布尔上下文里会对任何值求值。大多数检查不需要 == True 或 != None。

    Python — 假值（falsy）
    # 这些都是假值——在布尔上下文里相当于 False：
False
None
0          # 零整数
0.0        # 零浮点
""         # 空字符串
[]         # 空列表
{}         # 空字典
()         # 空元组
set()      # 空集合

# 其他所有值都是真值（truthy）
if []:                # False——空列表
    print("有元素")

items = [1, 2, 3]
if items:             # True——非空列表
    print("有元素")   # 会打印这个

    三元表达式（ternary expression）

    Python — 内联条件
    # Python 三元式：条件为真时的值 if 条件 else 条件为假时的值
x = 10
label = "偶数" if x % 2 == 0 else "奇数"
print(label)  # 偶数

# C++ 等价：label = (x % 2 == 0) ? "偶数" : "奇数";

    for 循环 — 遍历任何东西

    Python 的 for 循环遍历任何可迭代对象（iterable）——列表、字符串、range、字典、文件或自定义对象。没有 for (int i=0; i<n; i++) 这种基于索引的写法，用 range() 代替。

    Python — 基于 range 的 for 循环
    # range(stop) — 0 到 stop-1
for i in range(5):
    print(i)      # 0 1 2 3 4

# range(start, stop) — start 到 stop-1
for i in range(2, 6):
    print(i)      # 2 3 4 5

# range(start, stop, step)
for i in range(0, 10, 2):
    print(i)      # 0 2 4 6 8

# 倒数
for i in range(5, 0, -1):
    print(i)      # 5 4 3 2 1
    C++ 对比
    for i in range(n) 是 Python 版的 for (int i = 0; i < n; i++)。 for item in collection 类似 C++11 的范围 for：for (auto& item : collection)。 Python 的 range() 是惰性的——它不会真的创建一个数字列表，而是按需生成。类似 C++ 的生成器或惰性 view。

    遍历集合

    Python
    fruits = ["苹果", "香蕉", "樱桃"]

# 直接遍历——最简洁
for fruit in fruits:
    print(fruit)

# 带索引——用 enumerate()
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")    # 0: 苹果, 1: 香蕉, 2: 樱桃

# 并行遍历两个列表——用 zip()
prices = [3.5, 2.0, 8.0]
for fruit, price in zip(fruits, prices):
    print(f"{fruit}: ¥{price}")

# 遍历字典
ages = {"Alice": 30, "Bob": 25}
for name, age in ages.items():
    print(f"{name} 今年 {age} 岁")

    while 循环

    Python
    count = 0
while count < 5:
    print(count)
    count += 1    # 注意：Python 没有 ++ 或 -- 运算符！

# 读取直到哨兵值
total = 0
while True:
    val = int(input("输入数字（0 退出）："))
    if val == 0:
        break
    total += val
print(f"总计：{total}")
    Python 没有 ++
    Python 没有自增（++）或自减（--）运算符。用 x += 1 和 x -= 1。这是刻意的设计选择——++x 和 x++ 的差异被认为是困惑之源。

    break、continue、pass

    Python
    # break — 完全退出循环
for i in range(10):
    if i == 5:
        break
    print(i)          # 打印 0 1 2 3 4

# continue — 跳过当前迭代
for i in range(10):
    if i % 2 == 0:
        continue      # 跳过偶数
    print(i)          # 打印 1 3 5 7 9

# pass — 什么都不做（空代码块占位符）
for i in range(5):
    pass              # 语法需要，什么也不做

# 也用于空的类/函数体
def todo_function():
    pass              # 稍后实现

    循环的 else 子句 — Python 的独特之处

    Python 循环有一个可选的 else 子句，只有在循环没有碰到 break 正常结束时才会执行。C++ 里没有这个东西，但它对搜索模式确实很有用。

    Python — 循环的 else 子句
    target = 7
numbers = [1, 3, 5, 9, 11]

for n in numbers:
    if n == target:
        print(f"找到 {target}！")
        break
else:
    # 只有在从未执行 break 时才运行
    print(f"没找到 {target}")   # 会打印这个

# 等价的 C++ 写法需要一个 flag 变量——Python 避免了这一点

    列表推导式（list comprehension）

    列表推导式是 Python 最优雅的特性之一。它让你用一行可读的代码，通过表达变换和可选过滤来构建列表。如果你发现自己在写一个 for 循环，只是为了把结果塞进一个列表，推导式几乎总是更简洁的选择。

    Python — 推导式 vs. 普通循环
    # C++ 风格——构建一个平方数的 vector
# std::vector<int> squares;
# for (int i = 0; i < 10; i++) squares.push_back(i * i);

# Python 循环风格
squares = []
for i in range(10):
    squares.append(i ** 2)

# Python 推导式——同样的结果，一行搞定
squares = [i ** 2 for i in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
    Python — 带条件的推导式
    # 过滤：只要偶数的平方
even_squares = [x**2 for x in range(10) if x % 2 == 0]
# [0, 4, 16, 36, 64]

# 变换字符串
words = ["hello", "world", "python"]
upper = [w.upper() for w in words]
# ['HELLO', 'WORLD', 'PYTHON']

# 嵌套推导式——展平二维列表
matrix = [[1,2,3],[4,5,6],[7,8,9]]
flat = [n for row in matrix for n in row]
# [1, 2, 3, 4, 5, 6, 7, 8, 9]

    字典和集合推导式

    Python — 其他推导式形式
    # 字典推导式（dict comprehension）
words = ["cat", "elephant", "dog"]
lengths = {word: len(word) for word in words}
# {'cat': 3, 'elephant': 8, 'dog': 3}

# 集合推导式（set comprehension）——自动去重
nums = [1, 2, 2, 3, 3, 3]
unique_squares = {x**2 for x in nums}
# {1, 4, 9}

# 生成器表达式（generator expression）——惰性求值，不创建列表
total = sum(x**2 for x in range(1000))   # 注意没有 []！
    什么时候用推导式
    适合推导式的情形：你在通过变换或过滤一个现有集合来构建新集合，逻辑能舒适地放在一行里，意图清晰。如果每次迭代需要多条语句，或者嵌套很复杂，普通循环更易读。

    嵌套循环与条件

    Python — 乘法口诀表
    for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i}×{j}={i*j}", end="  ")
    print()   # 每行结束后换行

# 输出：
# 1×1=1  1×2=2  1×3=3
# 2×1=2  2×2=4  2×3=6
# 3×1=3  3×2=6  3×3=9
    Python — 嵌套条件（FizzBuzz）
    for n in range(1, 21):
    if n % 15 == 0:
        print("FizzBuzz")
    elif n % 3 == 0:
        print("Fizz")
    elif n % 5 == 0:
        print("Buzz")
    else:
        print(n)
