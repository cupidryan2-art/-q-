    第 3 章

    内置数据结构

    Python 内置四种集合类型：list、tuple、dict 和 set。 它们覆盖了你在 C++ 里会用 std::vector、std::unordered_map 或 std::set 做的绝大多数事情。 关于每种类型，最关键的问题是：它可变（mutable）还是不可变（immutable）？

    list — 有序可变序列

    list 是 Python 里用得最多的容器：有序、从零开始索引、可变。它最接近 std::vector<T>，但同一个 list 里可以存不同类型的对象。

    Python — 创建 list
    nums    = [1, 2, 3, 4, 5]
mixed   = [42, "hello", 3.14, True]   # 不同类型可以放在一起
empty   = []
from_range = list(range(5))           # [0, 1, 2, 3, 4]
repeated   = [0] * 5                  # [0, 0, 0, 0, 0]
    Python — 索引与切片
    fruits = ["苹果", "香蕉", "樱桃", "枣子"]

fruits[0]      # '苹果'  — 第一个元素
fruits[-1]     # '枣子'  — 最后一个元素
fruits[1:3]    # ['香蕉', '樱桃']  — 切片 [start, stop)
fruits[:2]     # ['苹果', '香蕉']
fruits[::2]    # ['苹果', '樱桃']   — 每隔一个取一个

# 就地修改
fruits[0] = "牛油果"          # 替换单个元素
fruits[1:3] = ["蓝莓"]        # 用更少的元素替换一个切片
    Python — list 方法（增删改查）
    nums = [3, 1, 4, 1, 5, 9]

# 添加
nums.append(2)           # 加到末尾：[3,1,4,1,5,9,2]
nums.insert(0, 99)       # 在索引处插入：[99,3,1,4,1,5,9,2]
nums.extend([6, 5])      # 追加多个元素：追加另一个 list

# 删除
nums.remove(1)           # 删除第一个值为 1 的元素
popped = nums.pop()      # 删除并返回最后一个元素
popped2 = nums.pop(0)    # 删除并返回索引 0 处的元素
del nums[2]              # 按索引删除

# 查找
nums.index(5)            # 第一次出现的索引（找不到抛 ValueError）
nums.count(1)            # 计算出现次数

# 排序
nums.sort()              # 就地排序（修改原 list）
nums.sort(reverse=True)  # 就地降序
sorted_copy = sorted(nums)   # 返回新的已排序 list，原 list 不变
nums.reverse()           # 就地反转

# 其他
len(nums)                # 元素数量
nums.copy()              # 浅拷贝
nums.clear()             # 清空 list
    C++ 对比
    • list.append(x) ≈ vec.push_back(x)
    • list.pop() ≈ vec.pop_back()——但 Python 的版本会返回被删除的值
    • list.insert(i, x) ≈ vec.insert(vec.begin()+i, x)
    • sorted(lst) ≈ 对副本执行 std::sort——Python 的排序是 Timsort，稳定且 O(n log n)
    • 和 std::vector 不同，Python list 是动态类型的——不需要模板参数

    用 key 函数排序

    Python
    words = ["banana", "fig", "apple", "elderberry"]

# 按长度排序
words.sort(key=len)
# ['fig', 'apple', 'banana', 'elderberry']

# 按最后一个字符排序
words.sort(key=lambda w: w[-1])

# 对字典列表按某个字段排序
people = [{"name": "Alice", "age": 30}, {"name": "Bob", "age": 25}]
people.sort(key=lambda p: p["age"])
# Bob 排在前面了

    tuple — 有序不可变序列

    tuple 像 list，但不可变——一旦创建就不能修改内容。在数据不应该变化的地方用 tuple：坐标、RGB 颜色、函数返回值、字典键。

    Python — tuple
    point  = (3, 7)
color  = (255, 128, 0)     # RGB——语义上是固定的
single = (42,)             # 单元素 tuple 需要拖尾逗号！
empty  = ()

# 不可变——下面这行会抛 TypeError：
# point[0] = 99

# tuple 解包（unpacking）
x, y = point               # x=3, y=7
r, g, b = color

# 函数返回多个值——其实就是一个 tuple
def min_max(nums):
    return min(nums), max(nums)   # 返回一个 tuple

lo, hi = min_max([3, 1, 4, 1, 5, 9])
print(lo, hi)   # 1 9

# 用 tuple 解包交换变量
a, b = 10, 20
a, b = b, a     # 不需要临时变量
    C++ 对比
    Python tuple 很好地对应 C++17 的 std::tuple 和结构化绑定： auto [lo, hi] = min_max(nums); 就是 lo, hi = min_max(nums) 的 C++ 版本。 Python tuple 可以用作字典键（因为它们可哈希且不可变）；std::vector 不能作为 std::unordered_map 的键，原因相同。

    dict — 键值映射

    Python 的 dict 是哈希表：查找、插入、删除的平均时间复杂度都是 O(1)。从 Python 3.7+ 开始，dict 保持插入顺序。

    Python — 创建和访问 dict
    person = {"name": "Alice", "age": 30, "city": "北京"}
empty  = {}
from_keys = dict(name="Bob", age=25)   # 关键字参数形式

# 访问
person["name"]          # 'Alice' — 键不存在会抛 KeyError
person.get("name")      # 'Alice' — 键不存在返回 None（安全）
person.get("email", "N/A")  # 'N/A' — 键不存在时返回默认值

# 修改
person["age"] = 31      # 更新已有键
person["email"] = "a@b.com"  # 添加新键
del person["city"]      # 删除键

# 成员测试
"name" in person        # True
"salary" in person      # False
    Python — 遍历 dict
    person = {"name": "Alice", "age": 30}

# 遍历键（默认）
for key in person:
    print(key)           # name, age

# 遍历值
for val in person.values():
    print(val)           # Alice, 30

# 遍历键值对
for key, val in person.items():
    print(f"{key}: {val}")

# 常用方法
list(person.keys())      # ['name', 'age']
list(person.values())    # ['Alice', 30]
list(person.items())     # [('name', 'Alice'), ('age', 30)]

# 合并两个 dict（Python 3.9+）
defaults = {"color": "red", "size": "M"}
custom   = {"size": "L", "weight": 1.5}
merged   = defaults | custom   # custom 的值在冲突时优先
# {'color': 'red', 'size': 'L', 'weight': 1.5}

    字典推导式

    Python
    words = ["apple", "fig", "banana"]

# 把每个单词映射到它的长度
lengths = {w: len(w) for w in words}
# {'apple': 5, 'fig': 3, 'banana': 6}

# 反转字典（交换键和值）
original = {"a": 1, "b": 2, "c": 3}
inverted = {v: k for k, v in original.items()}
# {1: 'a', 2: 'b', 3: 'c'}
    C++ 对比
    Python dict ≈ std::unordered_map（哈希表，平均 O(1)）。 Python dict 保持插入顺序；std::unordered_map 不保证。 如果你需要按键排序的遍历，用 sorted(d.keys())——C++ 里用 std::map 来对应。

    set — 无序唯一值集合

    set 存储唯一值，不保证顺序。成员测试 O(1)，支持完整的数学集合运算。

    Python — set
    primes = {2, 3, 5, 7, 11}
empty  = set()    # 注意！不是 {}——那会创建一个空 dict！

# 去重——把有重复的 list 转成 set
nums = [1, 2, 2, 3, 3, 3, 4]
unique = set(nums)    # {1, 2, 3, 4}

# 成员测试（O(1)）
3 in primes           # True
4 in primes           # False

# 修改
primes.add(13)
primes.discard(2)     # 存在就删，不存在不报错
primes.remove(3)      # 删除——不存在抛 KeyError

# 集合运算
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a | b     # 并集：        {1, 2, 3, 4, 5, 6}
a & b     # 交集：        {3, 4}
a - b     # 差集：        {1, 2}  — 在 a 不在 b
a ^ b     # 对称差：      {1, 2, 5, 6}  — 在其中一个里，但不在两个里

a.issubset(b)       # a ⊆ b？
a.issuperset(b)     # a ⊇ b？

    可变性详解

    这个区别非常关键，尤其是当你开始把容器传进函数时。

    类型可变？可哈希？典型用途
    list是 ✓否需要修改的有序集合
    tuple否是 ✓固定记录；dict 键；返回值
    dict是 ✓否键值查找
    set是 ✓否唯一值；成员测试
    frozenset否是 ✓不可变集合；可作 dict 键
    str否是 ✓文本
    Python — 函数里的变异（重要！）
    def double_all(lst):
    for i in range(len(lst)):
        lst[i] *= 2   # 变异了原始 list！

numbers = [1, 2, 3]
double_all(numbers)
print(numbers)   # [2, 4, 6] — list 被就地修改了

# 为了避免变异，传入副本：
double_all(numbers[:])   # 或者 numbers.copy()
    C++ 对比
    C++ 里，按值传递会复制容器；按引用或指针传递允许变异。 Python 里，所有参数都按对象引用传递。可变对象（list、dict、set）在函数内部可以被变异。不可变对象（int、str、tuple）不能。这类似于 C++ 的 const& vs 可变 &，但是隐式的——Python 没有 const 关键字来在类型层面强制不可变性。

    嵌套结构 — DataFrame 的前奏

    真实数据很少是扁平数组。字典列表是 Python 数据工作里最常见的模式之一——它直接对应一张电子表格或数据库表，也正是 pandas DataFrame 的内部表示。

    Python — 字典列表（类似表格的数据）
    students = [
    {"name": "Alice",   "score": 92, "grade": "A"},
    {"name": "Bob",     "score": 78, "grade": "C"},
    {"name": "Charlie", "score": 85, "grade": "B"},
]

# 访问一个字段
students[0]["name"]    # 'Alice'

# 遍历行
for s in students:
    print(f"{s['name']}: {s['score']}")

# 用推导式过滤
honor_roll = [s for s in students if s["score"] >= 90]

# 提取一列
names  = [s["name"]  for s in students]
scores = [s["score"] for s in students]

# 按分数排序
students.sort(key=lambda s: s["score"], reverse=True)
