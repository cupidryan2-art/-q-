    Python 基础语法

    Python 里的一切都是对象（object）——整数、字符串、函数，甚至 None，它们全都活在堆上，有各自的类型、值和引用计数。 这一章打地基：变量、类型、字符串和运算符。

    变量与动态类型

    Python 变量是一个绑定到对象的名字。赋值创建这个绑定，并不像 C++ 声明那样分配有类型的存储空间。

    Python
    x = 42          # x 现在是指向整数对象 42 的名字
x = "hello"     # x 现在指向字符串——完全合法
x = [1, 2, 3]   # 现在是个 list

# 多重赋值
a = b = c = 0   # 三个名字都指向同一个整数 0

# 元组解包（tuple unpacking）——不需要临时变量就能交换
a, b = 10, 20
a, b = b, a     # 现在 a=20, b=10
    C++ 对比

    C++ 里，int x = 5; 保留了一个有类型的内存槽。之后再赋字符串给它是编译报错。

    Python 里，x = 5 在堆上创建一个整数对象，让 x 指向它。x = "hello" 创建字符串对象，重新指向它。整数对象依然存在，直到引用计数降到零被 GC 回收。

    把 Python 变量想成 std::any，或者一个可以重新指向任意类型的有类型指针。

    命名规范

    Python 用 snake_case 命名变量和函数（不用 camelCase）。 以 _ 开头的名字按惯例是"私有"的。像 __dunder__（双下划线）这样的名字是 Python 自身协议保留的。

    核心数据类型

    Python — 五种标量类型
    age    = 30          # int   — 任意精度整数，不会溢出
price  = 19.99       # float — 64 位 IEEE 754，等同于 C++ double
name   = "Alice"     # str   — 不可变的 Unicode 文本
active = True        # bool  — True 或 False（首字母大写！）
empty  = None        # NoneType — Python 的 null / void 等价物
    值得关注
    Python 的 int 有任意精度——它能存下内存允许的任何大数。 不存在 int32_t 溢出：2 ** 1000 是一个合法的整数。 如果需要定宽整数来和 C 代码对接，用 ctypes 或 struct 模块。
    C++ 对比
    • Python int ≈ C++ 大数（不是 int32_t）
    • Python float ≈ C++ double
    • Python bool：True/False 其实是 int 的子类——True == 1，False == 0
    • Python None ≈ C++ nullptr 或 std::nullopt——但它是一个真实的对象，有自己的类型

    数字字面量

    Python
    # 整数进制
dec = 255
hex_val = 0xFF     # 十六进制
oct_val = 0o377    # 八进制
bin_val = 0b11111111  # 二进制——以上都等于 255

# 数字里的下划线（增加可读性，Python 忽略它）
million = 1_000_000
pi = 3.141_592_653

# 字符串分隔符——单引号或双引号都行
s1 = 'hello'
s2 = "world"
s3 = '''多行
字符串'''          # 三引号可以跨行

    运行时类型检查

    Python
    x = 42

# type() 返回精确的类型对象
print(type(x))           # <class 'int'>
print(type(x) is int)    # True

# isinstance() 检查类型或其父类——优先用这个
print(isinstance(x, int))         # True
print(isinstance(x, (int, float))) # True——可以同时检查多种类型

# bool 是 int 的子类——isinstance 会捕获到它
print(isinstance(True, int))   # True  ← 可能让你意外
print(type(True) is int)       # False——精确类型检查能区分它们
    最佳实践
    用 isinstance() 而不是 type() == 检查。它尊重继承关系，是 Python 惯用写法。你在库代码里到处都能看到它。

    类型转换

    Python — 显式类型转换
    int("42")         # 42    — 字符串转整数
float("3.14")     # 3.14
str(100)          # "100" — 整数转字符串
bool(0)           # False — 零是假值（falsy）
bool("hello")     # True  — 非空字符串是真值（truthy）

int(3.9)          # 3     — 截断，不是四舍五入
round(3.9)        # 4     — 四舍五入到最近整数

    字符串操作

    f-string（格式化字符串字面量）

    Python 3.6 引入的 f-string 是现代嵌入表达式的方式。在字符串前加 f，把任何表达式放进 {} 里就行。

    Python — f-string
    name  = "Alice"
score = 98.6

# 基本嵌入
print(f"你好，{name}！")               # 你好，Alice！

# {} 里可以写表达式
print(f"分数：{score:.1f}%")           # 分数：98.6%
print(f"2 + 2 = {2 + 2}")             # 2 + 2 = 4
print(f"大写：{name.upper()}")         # 大写：ALICE

# 对齐与填充
print(f"{'left':<10}|")               # 'left      |'
print(f"{'right':>10}|")              # '     right|'
print(f"{42:05d}")                     # '00042'
print(f"{3.14159:.2f}")               # '3.14'

# 调试简写（Python 3.8+）
x = 42
print(f"{x=}")                         # x=42
    C++ 对比
    f-string 本质上是 Python 版的 C++20 std::format("{}", value) 或旧版 sprintf，但是内联的，在运行时求值。和 printf 不同，格式说明和表达式在同一个字符串字面量里。

    常用字符串方法

    Python — 字符串方法（字符串是不可变的——所有方法返回新字符串）
    s = "  Hello, World!  "

s.strip()           # "Hello, World!"   — 去掉首尾空白
s.lower()           # "  hello, world!  "
s.upper()           # "  HELLO, WORLD!  "
s.replace("World", "Python")  # "  Hello, Python!  "
s.split(",")        # ['  Hello', ' World!  ']
", ".join(["a", "b", "c"])    # "a, b, c"

s.startswith("  H")  # True
s.endswith("  ")     # True
s.strip().find("World")  # 7  — 第一个匹配的索引，-1 表示未找到
"hello" in s.lower() # True  — 成员测试

    字符串切片（slicing）

    Python 的切片语法 s[start:stop:step] 是最具表现力的特性之一。所有索引都是可选的。

    Python — 切片
    s = "Hello, World!"
#    0123456789...

s[0]       # 'H'          — 从头部索引
s[-1]      # '!'          — 从尾部索引
s[7:12]    # 'World'      — [start, stop)
s[:5]      # 'Hello'      — 从头到索引 5
s[7:]      # 'World!'     — 从索引 7 到末尾
s[::2]     # 'Hlo ol!'    — 每隔一个字符（步长=2）
s[::-1]    # '!dlroW ,olleH'  — 反转字符串
    C++ 对比
    Python 切片类似于 std::string::substr(start, length)，但更强大：负索引、步长参数，而且 stop 索引是排他的（就像 STL 迭代器）。注意 Python 字符串是不可变的——s[0] = 'h' 会抛 TypeError。要修改，用 s = 'h' + s[1:] 或先转成 list。

    运算符

    算术运算符

    Python
    10 + 3    # 13   — 加
10 - 3    # 7    — 减
10 * 3    # 30   — 乘
10 / 3    # 3.333...  — 真除法（总是返回 float）
10 // 3   # 3    — 整除（向下取整）
10 % 3    # 1    — 取模（求余数）
2 ** 10   # 1024 — 幂运算（不需要 std::pow）
    注意
    Python 3 里，/ 永远是真除法——7 / 2 是 3.5，不是 3。 用 // 做整除。这和 C++ 不同——C++ 里整数操作数的 7 / 2 == 3。

    比较与逻辑运算符

    Python
    # 比较
5 == 5     # True
5 != 4     # True
5 > 3      # True
5 <= 5     # True

# Python 支持链式比较——非常易读
1 < x < 10       # 等价于 x > 1 and x < 10
0 <= age < 120   # 一个表达式搞定范围检查

# 逻辑运算符——用英文关键字，不用符号
True and False   # False
True or False    # True
not True         # False

# 身份 vs 相等
a = [1, 2, 3]
b = [1, 2, 3]
a == b     # True  — 值相同
a is b     # False — 内存里是不同对象
a is None  # False
x = None
x is None  # True  — 检查 None 用 is，不用 ==
    C++ 对比
    Python 的 and/or/not 等价于 C++ 的 &&/||/!。Python 的 is 运算符检查指针相等（类似比较两个裸指针的地址），而不是指针指向的值。

    输入与输出

    Python — print()
    print("Hello")                    # Hello\n（默认有换行）
print("a", "b", "c")             # a b c  （默认空格分隔）
print("a", "b", "c", sep="-")    # a-b-c
print("没有换行", end="")          # 去掉末尾的 \n
print()                           # 只打印一个换行

# 打印多种类型——不需要手动转换
x = 42
print("值：", x, "类型：", type(x))  # 值：42 类型：<class 'int'>
    Python — input()
    name = input("请输入你的名字：")   # 总是返回 str
age  = int(input("请输入你的年龄："))   # 如果需要整数，立即转换

print(f"你好 {name}，你今年 {age} 岁。")
    关键点
    input() 总是返回字符串。如果你想要数字，记得套一层：int(input(...)) 或 float(input(...))。忘记这一点是最常见的初学错误之一。
